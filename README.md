# Economy
Spigot plugin for handling in-game Vault Economy.

----

## Dependencies
- Vault: https://www.spigotmc.org/resources/vault.34315/

## Features
- Admin and Player commands
- Cross-Server
- MySQL Support
- Huge Configuration
